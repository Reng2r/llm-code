# LLM Handwritten Code Lab

本项目用于记录我在学习大语言模型（LLM）过程中的手撕代码与实现练习。

## 项目简介

在学习 LLM 的过程中，仅仅停留在概念理解往往不够。  
因此，我希望通过“手撕代码”的方式，把常见模块、核心算法和基础组件真正写出来、跑起来、讲清楚。

本仓库主要用于：

- 记录 LLM 学习过程中的代码实现
- 练习常见模块的手写与复现
- 加深对模型原理、张量计算与工程细节的理解
- 为后续面试准备和项目实践积累素材

## 当前内容

本项目会逐步补充与整理以下内容：

- Softmax
- Multi-Head Attention
- LayerNorm / RMSNorm
- RoPE
- MLP / FFN
- Transformer Block
- KV Cache
- Sampling（Greedy / Top-k / Top-p）
- 常见训练与推理小模块

## 项目目标

希望通过持续手写与复现，达到以下目标：

1. 理解 LLM 常见模块的输入输出与计算流程  
2. 熟悉 PyTorch 张量操作与代码实现方式  
3. 能够在面试中独立写出核心模块的简化版本  
4. 形成一套属于自己的 LLM 学习与复习资料

## 适合人群

本仓库适合：

- 正在学习大语言模型基础的同学
- 准备算法 / 大模型相关面试的同学
- 想通过手写代码理解 Transformer 与 LLM 组件的人

## 持续更新

本项目会随着学习进度不断更新。  
如果某些实现还不够完善，说明我也还在继续学习和打磨。

## 说明

本仓库以学习、理解和练习为主，侧重“从原理到代码”的过程记录。  
部分实现可能会采用教学化、简化版写法，以便更清楚地展示核心逻辑。
